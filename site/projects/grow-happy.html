Grow Happy is a self reflection tool that lets users evaluate key areas of their life, set goals for lacking areas. Evaluations are aggregated into scores. Users can visualize their hapiness scores with charts and can analyize how scores evolve over time.

Features

  • front-end app built with a javascript framework: Vue.js
  • data mapped on charts with a 3rd party library: Chart.js
  • user auth forms (register, login) with data validation
  • user authentication system using Firestore Authentication
  • data persistance using Google Firestore Realtime Database
  • responsive web design
  • SOLID app architecture
