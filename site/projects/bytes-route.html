Bytes Route is a free SaaS for building and deploying product tours for websites and web-applications, without any coding background. It consists of a Vue.js dashboard, a Chrome extension used as the tour builder, a script for integrating with your website, a Node.js backend and a presentation website

A view of the dashboard app

Sub-projects

Bytes Route consists of several sub-projects:

Website

A presentation website containing info about the product, built with vanilla JS, pug and stylus. Features a main page, a blog and docs.

https://www.bytesroute.com
Dashboard app

An front-end app built with Vue.Js where users authenticate, manage created product tours, manage team members, custom styles and more.

https://www.bytesroute.com/app
Tour creator

This is a Chrome extension that Bytes Route clients use to build, edit and test tours for their websites. This was built with Vuejs, vanillaJS, pug and stylus.

Chrome extension
Backend (API)

Built with Nodejs, Express, MongoDB and Mongoose. Responsible for CRUD operations called from the Dashboard app and the Chrome extension

Script

The script is written in VanillaJS. It is responsible for tour creation and tour running. Bytes Route users receive a script with a custom id that they can integrate in their apps to enable tours running on Bytes Route..

Features

  • large-scale Vue.js app
  • back-end API using Node.js and Express
  • user-generated content: product tours
  • Chrome extension addon: used to build tours
  • user authenthication system
  • team management system
  • theme style editing, edit the color scheme of the tour pop-ups to match your brand identity
  • multilanguage feature, set your tours to run in multiple languages
  • unit tests
