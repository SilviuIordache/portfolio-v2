This was my bachelor project. This game was built using the Game Maker Studio 2 engine. It features a procedurally generated game level in each session, making no 2 playthroughs the same.

It is a 2D top down game where you take control of a space marine who is fighting against mobs of zombie aliens on an oxigen depleted space-station

Controls: WASD + left click

Objectives

  • shoot all the monsters in each room to open the doors holding back oxygen
  • upgrade your weapons with power-ups
  • see how long you can last on the space station
